Building a Graffiti App

Graffiti is an architecture that makes it possible to build malleable and interoperable social applications using nothing but standard front-end web development tools. This guide explains how to build these web apps with the Graffiti client Javascript library and it's plugin for the Vue.js web framework.

If you're not familiar with Vue, make sure to check out the Vue docs before getting started with this guide. Note that there is nothing inherently tying Graffiti to Vue, it just happens to be the first framework we added support for. If you're interested in make a Graffiti plugin for a different web framework, please contact us or make a pull request.

This guide includes a number of interactive demos, like the one below. The included source code can be modified and the demo will update dynamically. Truly malleable social media!
